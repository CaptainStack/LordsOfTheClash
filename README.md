# LordsOfTheClash
A 2D clash tactics game that features huge battles.

The following outline is a work in progress and subject to change.

# Supported Platforms
* Microsoft Windows
* MacOS
* Linux
* Xbox One
* Playstation 4
* Nintendo Switch
* iPad

# Supported Input devices
* Touch
* Controller
* Mouse/keyboard

# Gameplay modes
## Single player
* Story mode
* Arcade / horde

## Multiplayer
* Shared screen versus
* Online competitive matchmaking
